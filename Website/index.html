Recommended Processing Workflow:

  1. Preprocessing Data
    1. Raw Data Plotting (for frequent comparison to current data)
    2. Artifact Removal (ICA very applicable to EEG)
    3. Eye Blink Removal (using ICA and EOG channels near the eyes)
    4. Filtering: Bandpass filtering can be performed initially or here
    5. Epoching: Segment the data into the sections we want
    6. Baseline Correction: Determine how we want to apply a baseline to our different "events"
  2. Data Analysis
    1. Time-Frequency Analysis
    2. Power Spectral Density (PSD) Calculation
    3. Current Source Density (CSD) Calculation
    4. Source Localization
    5. Statistical Analysis
  3. Visualization
    1. Raw Data Plotting
    2. ERP/ERF Plotting
    3. Topographic Maps
    4. Electrode PSD
    5. PSD Plotting
    6. CSD Visualizations
    7. Time-Frequency Maps

Configuration

To effectively process data using MNE Python in the exact same way as us, please ensure your Environment contains the following:

  • MNE python version:
  • Python Version: 3.12.8
  • Optional: Additional libraries for advanced analysis (e.g., scikit-learn, matplotlib)
If you find you cannot duplicate our results, you may need to manually specify these dependencies, or create an isolated virtual environment (provided at: HTML link)
