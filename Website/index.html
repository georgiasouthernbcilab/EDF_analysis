It is recommended that you complete the project in the following order:

  1. Install Operating Systems
    1. Windows Server
    2. Windows 10
    3. Ubuntu
    4. Update all operating systems (at home, it takes a while)
  2. Windows 10
    1. Disable UAC (If needed for your startup method)
    2. Install Wamp
    3. Configure Wamp
    4. Install Joomla
    5. Customize your Joomla (at home, no instructions as everyone's will be different!)
    6. Backup Joomla
    7. Open Firewall for outside access
  3. Ubuntu
    1. Install XAMPP
    2. Restore Joomla (windows 10 PC must be open with Wamp server running, the correct ports open, and you must know the IP address to navigate to the WAMP server, and the backup location on the Wamp server.)
  4. Windows Server
    1. Set static IP
    2. Add services
    3. Configure DHCP scope
  5. Active Directory / Windows Server
    1. Add Windows to AD
    2. Add Ubuntu to AD

Configuration

The project will require the following to be installed, configured, and operational:

  • Windows
    • WAMP with website (Joomla as a CMS with customizations)
    • Connected to AD
  • Ubuntu
    • XAMPP with WAMP backup
    • Connected to AD
  • Windows Server
    • Active Directory
    • DHCP
    • DNS
