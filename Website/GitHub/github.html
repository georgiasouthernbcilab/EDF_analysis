    To fix, do:
  1. Save working changes in other directory
  2. Delete local repository and re-download from GitHub
  3. Move your modified files back into the repo & push

Oftentimes, after accidentally adding a large file, the changes are still in the local .git file, and will cause continuous issues until resolved. You can attempt to clean the working tree / stash changes aswell, but this is sometimes the most straight forward.
